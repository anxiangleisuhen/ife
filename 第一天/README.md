题目：

我们假设现在要为自己创建一个在线简历，那简历都包含什么呢？给一些参考：

包括“简历”两个字
你的姓名，有可能只有姓名，有可能是 姓名：张三 这样的模式
你的各种联系方式，比如：手机：18600000000，邮箱：joinefe@baidu.com
你的学历，比如：本科：百度前端学院
项目经验，比如：项目A：项目时间：2017.1-2017.8，项目描述：在项目A中负责了xxxxx

注意点：
①空格符&nbsp;是打在<p>里边的，在外边不会有任何效用；
②一些特殊字符如&copy;的分号可有可无，若是英文状态下便不会显示出来，直接显示特殊字符；
  若在中文状态下则会打出“；”的符号。
